HealthCare++


The purpose of this Project is to create a application for hospitals to showcase the Telecare concept for their patients. This application enables the Patient to get attached to the Hospital when they are outside like Home/Office. This Application serves all the basic needs of the patients like maintaining consultation reports, Booking appointments with doctor, paying bills online, and also efficient search for hospitals and near by pharmacy when needed. And it also has healthy notification facility to alert the patients about important schedules and remainders. Thus this project provides a complete Hospital Telecare facility to patients.

This App is a Part of CSC COIN PROJECT
